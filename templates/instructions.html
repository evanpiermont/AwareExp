This is an experiment! Your payment will be determined in part by your preformance on the following task.

You will be shown {{handsize}} cards. The task invloves identifing sets of cards.
Each card has 3 properties: (i) a symbol, (ii) a number, and, (iii) a color.
Each property takes 4 values. Symbol: Triangle, Square, Circle, or Star. Number: 1, 2, 3, or 4. Color: Purple, Red, Orange, or Blue.
A set is any 3 cards in which all 3 properties are either all the same or all different. The next pages have some examples.
For example, the following 3 cards form a set:
Symbol: each card has a differnt symbol. Number: each card has the same number, 4. Color: each card has the same color, orange.
The following 3 cards also form a set:
Symbol: each card has the same symbol, a triangle. Number: each card has a different number. Color: each card has a different color.
On the other hand, the following 3 cards do not form a set:
This is becuase 2 of the cards have a number of 1, but the third card has a number of 3. (Also, the second and third cards have circles, where the first has a triangle.)
You will be shown 9 cards; you will have {{rndtime}} seconds to click on the cards to form sets. The order the cards are clicked does not matter. You can click a selected card to un-select it. If you select 3 cards that form a set, the set will be registered on the side bar.
If the 3 cards do not form a set, you will incur a 2 second penalty. If you select the same set more than once, you will incur a 2 second pelanty.
You will receive {{payment}} cents per set found. After the time ends, you will be asked some survey questions. After these have been completed, you will receive an M-TURK code to enter on your M-TURK account.
